Table

a collection class for Players, represents the game table a table is an array of arrays where each array has two elements: the seat position of the player, and the player object itself in the first “seat” of the Array is in play the seat_counter keeps track of the numer of seats at the table

  @seat_counter = @seat_counter + 1 
  @seats.push([@seat_counter, player])
  self

def to_s
  @seats.sort_by {|position, player| position}.collect do |position, player|
    "Seat #{position}: #{player.to_s}"
  end.join("\n")
end
